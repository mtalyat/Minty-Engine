
DESCRIPTOR TYPES:
	VK_DESCRIPTOR_TYPE_UNIFORM_BUFFER
Data consists of the bytes of the uniform buffer object struct.

	VK_DESCRIPTOR_TYPE_COMBINED_IMAGE_SAMPLER
Data consists of an array of IDs to each texture to be loaded.
